# DLE-DG-Dataset
The paper of "DLE-DG: Decoupling Location and Environment Representations for Domain Generalization in RFID Fingerprint Indoor Localization" 's dataset.
The dataset is represented as a tensor of size nx280, where 'n' represents the total number of samples in the dataset. The datasets is stored in the Releases DLE-DG dataset, data.pt and dataLabels.pt are the data sets and labels, respectively. The label is [location,environment].
